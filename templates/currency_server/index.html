Currency server

  • Average exchange rate by the currency code and the date (formatted YYYY-MM-DD):
    http://localhost:{{ port }}/exchanges/one-day-rate/<code>/<date>
    http://localhost:{{ port }}/exchanges/one-day-rate/GBP/2023-04-21
  • Max and min average exchange rate by the currency code and the number of last quotations N (N ≤ 255):
    http://localhost:{{ port }}/exchanges/maximum-and-minimum/<code>/<N>
    http://localhost:{{ port }}/exchanges/maximum-and-minimum/GBP/23
  • The major difference between the buy and ask rate by the currency code and the number of last quotations N (N ≤ 255)
    http://localhost:{{ port }}/buy-and-sell-rates/<code>/<N>
    http://localhost:{{ port }}/buy-and-sell-rates/GBP/23
